== RUNNING THE WEBSITE WITH A FUNCTIONING DATABASE ==

XAMPP control panel is necessary to create a localhost server & run the PHP files.
if you decide to, please make sure Apache and MySQL are running.

Look for the htdocs folder under XAMPP, create a new folder inside it, and move all the files there.
then type localhost/ in your internet browser and select the folder where all the files are.
